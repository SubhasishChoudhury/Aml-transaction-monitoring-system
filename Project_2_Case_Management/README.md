# AML Alert Prioritization & Case Management System

## Project Overview

This project simulates an AML (Anti-Money Laundering) alert investigation workflow. Suspicious alerts are extracted from transaction monitoring data, prioritized using risk scores, converted into investigation cases, assigned to analysts, and evaluated for SAR decisions.

## Features

- Alert Prioritization Engine
- Case ID Generation
- Investigator Assignment
- Case Status Tracking
- SAR Decision Engine
- AML Summary Metrics
- Exploratory Data Analysis (EDA)

## Key Results

- Total Alerts: 20,276
- Critical Alerts: 2,861
- High Priority Alerts: 5,787
- SAR Required Cases: 2,861
- SAR Review Cases: 5,787
- No SAR Required Cases: 11,628
